# zanxpk

متجر تطبيقات وألعاب أندرويد + متجر منتجات رقمية، بملف PHP واحد (`index.php`) مع لوحة إدارة كاملة.

## المميزات
- نشر وإدارة تطبيقات/ألعاب أندرويد (أيقونة، صور، فيديو، صلاحيات، سجل تحديثات).
- توليد محتوى التطبيق بالذكاء الاصطناعي (الوصف، SEO، الأيقونة) من اسم التطبيق فقط، عبر OpenRouter.
- نظام صلاحيات مفاتيح OpenRouter متعدد مع تبديل تلقائي عند الفشل.
- صفحة تحميل مخصصة لكل تطبيق مع عداد انتظار، زر اشتراك بقناة تيليجرام، وصفحة شكر بعد التحميل مليئة بمساحات إعلانية.
- متجر منتجات رقمية مستقل عن متجر التطبيقات.
- تصفح وتحميل بدون تسجيل دخول؛ تسجيل الدخول مطلوب فقط للشراء والمفضلة وسجل الطلبات ولوحة الإدارة.
- بوت تيليجرام مستقل لتصفح أحدث التطبيقات والألعاب والبحث عنها.
- SEO: عناوين/أوصاف Meta فريدة لكل صفحة، Open Graph، JSON-LD، خريطة موقع، صفحات محتوى إضافية (دليل التحميل، أفضل التطبيقات).
- دعم Google AdSense (وسم التحقق + سكربت الإعلانات) وملف ads.txt قابلين للتهيئة من لوحة الإدارة.
- ترجمة تلقائية للموقع حسب لغة جهاز الزائر (Google Translate)، قابلة للتعطيل من لوحة الإدارة.
- إعلانات شبكة عامة (MoneyTag) قابلة للتفعيل/التعطيل من الإدارة، مع حماية من اهتزاز التخطيط (CLS).
- تسجيل دخول عبر Google OAuth وتيليجرام.
- نظام حماية متقدم: حماية من تخمين كلمات المرور (قفل تلقائي بعد عدد محاولات فاشلة قابل للتهيئة)، حظر/فك حظر عناوين IP يدوياً، تشديد جلسات تسجيل الدخول (HttpOnly/SameSite)، رؤوس أمان HTTP، وحماية ملفات الإعداد وقاعدة البيانات من الوصول المباشر — كل ذلك قابل للمتابعة والتحكم من تبويب "الحماية والأمان" في لوحة الإدارة.

## التشغيل محلياً
```bash
cp config.sample.php config.php
# عدّل القيم في config.php (قاعدة البيانات، البريد الإداري، ...)
php -S localhost:8000
```

## بوت تيليجرام
`telegram_bot.php` يوفر واجهة بوت تيليجرام مستقلة مرتبطة بنفس قاعدة البيانات، لتصفح أحدث التطبيقات والألعاب والبحث عنها وعرض المنتجات.

## قاعدة البيانات
يدعم SQLite (للتجربة المحلية) أو MySQL (للإنتاج)، يتم إنشاء/تحديث الجداول تلقائياً عند أول تشغيل (`migrate()`).
