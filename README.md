# Yassota — منصة متجر + ربح نقاط (ملف واحد)

منصة عربية متكاملة في ملف **`index.php`** واحد (HTML + CSS + JS + PHP + قاعدة بيانات).

## المميزات
- 🗄️ **قاعدة بيانات ذاتية** (SQLite) تُنشأ تلقائياً من نفس الملف — لا حاجة لملفات أخرى.
- 👑 **لوحة إدارة كاملة** تظهر فقط لإيميل المالك (`ADMIN_EMAIL`).
- 🛍️ **منتجات** كروت احترافية، الأحدث أولاً، أيقونة/وسم/خصم، طلب شراء → موافقة الإدارة.
- 💳 **محفظة ورصيد** بعملة `Yassota` + شحن (USDT / شام كاش / BTC) → موافقة الإدارة.
- 💸 **سحب الأرباح** بحد أدنى 25$ عبر شام كاش / USDT / بيتكوين.
- 🎯 **كابتشا محلية** (أرقام) لربح النقاط فوراً.
- 📋 **مهام يومية** (زيارة رابط 15 ثانية ثم استلام النقاط) مع تحقق.
- 📺 إعلانات **Monetag** (95% للموقع / 5% للمستخدم).
- 🔐 **تسجيل دخول جوجل** (OAuth) + جلسة أسبوع، مع وضع دخول تجريبي بالإيميل.
- 🤖 قسم **بوت تيليجرام** للربح بالنقاط.
- 🔒 صفحات **سياسة الخصوصية / الشروط** + موافقة كوكيز لمرة واحدة.
- 🧭 شريط جانبي منسدل احترافي + أزرار سفلية + SEO قوي + Lazy loading بشعار وبنر.

## التشغيل
ارفع `index.php` على أي استضافة **PHP 7.4+** تدعم `PDO SQLite`. سيُنشئ `yassota.sqlite` تلقائياً.

### الإعداد (أعلى الملف)
| ثابت | الوظيفة |
|------|---------|
| `ADMIN_EMAIL` | إيميل المالك الذي يرى لوحة الإدارة |
| `GOOGLE_CLIENT_ID` / `GOOGLE_CLIENT_SECRET` | تفعيل دخول جوجل الحقيقي |
| `MONETAG_ZONE_ID` | معرّف إعلانات Monetag |
| `COINS_PER_USD` / `MIN_WITHDRAW_USD` | نسبة التحويل والحد الأدنى للسحب |

> بدون بيانات جوجل يعمل **وضع دخول تجريبي بالإيميل**: ادخل بإيميل المالك للوصول للوحة الإدارة.
